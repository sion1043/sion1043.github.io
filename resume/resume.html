1. 간략 소개

대학 시절 진행한 캡스톤 디자인 프로젝트에서 AI SaaS를 접목하여 데이터를 분석하고 시각화하는 과정을 경험하며 데이터의 무한한 가능성을 발견했습니다. 단순한 기술 구현을 넘어, 파편화된 데이터가 정교한 분석을 통해 비즈니스 의사결정의 핵심 근거로 변모하는 가치를 목격한 후 데이터 전문가로서의 길을 확신하게 되었습니다. 이후 정보처리기사, SQLD 등의 자격증 취득과 함께 MS Data School을 포함한 1,100시간 이상의 실무 교육을 이수하며 전문성을 다졌습니다. 저는 기술적 숙련도와 비즈니스적 통찰력을 바탕으로 기업의 데이터를 가치 있는 자산으로 전환하는 '즉시 전력감' 데이터 인재입니다.

3. 성격의 장·단점

저의 가장 큰 강점은 데이터의 신뢰성을 끝까지 책임지는 꼼꼼함과 근본적인 문제 해결력입니다. 작은 오류가 분석 전체를 왜곡할 수 있기에, 저는 전처리 단계에서 예외 처리 로직 설계에 가장 많은 공을 들입니다. 실제 '인천공항 올인원 서비스' 프로젝트 당시, API 호출 지연 문제를 단순히 수집 주기를 늘려 해결하는 대신 정교한 재시도(Retry) 로직과 모니터링 시스템을 구축하여 파이프라인의 안정성을 99.9%까지 확보한 성과가 있습니다.

반면 완벽주의 성향으로 인해 배포 전 검토 시간이 다소 길어지는 면이 있으나, 이를 보완하기 위해 '우선순위 기반 의사결정 원칙'을 수립했습니다. '비즈니스 가치'와 '기술적 구현 가능성'이라는 명확한 기준을 바탕으로 팀 내 의견을 조율하고 의사결정 속도를 높임으로써, 업무의 완성도와 효율성 사이의 최적의 균형을 유지하고 있습니다.

4. 직무 경험 및 프로젝트

신입이지만 실무에 준하는 End-to-End 데이터 프로젝트를 주도하며 현업 투입 시 즉각적인 성과를 낼 수 있는 역량을 쌓았습니다. Azure 환경에서 데이터 수집부터 시각화까지 이어지는 전체 파이프라인을 직접 구축하고 운영해 본 경험은 저만의 독보적인 강점입니다.

인천공항 출국 전 종합 서비스 (ROAD 2 AIR)

Azure Function App과 Stream Analytics를 활용해 실시간 공공 데이터를 수집하고 전처리하는 파이프라인을 구축했습니다. Power BI Direct Query 연동을 통해 사용자에게 지연 없는 실시간 대시보드를 제공하며 엔지니어링 역량을 증명했습니다.

점핏 채용 지원 서비스 (DrawBridge)

기존 채용 시장의 주관적 평가 한계를 극복하기 위해 Azure OpenAI(GPT-4.1)와 Cosmos DB를 연동한 RAG 기반 숙련도 진단 시스템을 설계했습니다. 벡터 유사도 측정을 통해 객관적인 분석 지표를 생성하는 성과를 거두었습니다.

Steam 게임 성공요인 분석

Databricks를 활용한 메달리온 아키텍처 기반의 대규모 데이터 처리 시스템을 운영했습니다. 통계적 검정과 텍스트 마이닝을 통해 '리뷰 수정률'과 같은 새로운 KPI를 발굴하며 게임사에 실질적인 마케팅 인사이트를 제공했습니다.

5. 지원동기 및 입사 후 포부

데이터가 비즈니스의 생존과 성장을 결정짓는 핵심 자산이 된 지금, 저는 단순한 기술자가 아닌 기업의 전략적 파트너로서 기여하고자 합니다. 제가 보유한 Python, SQL, 클라우드 기술력은 귀사의 방대한 데이터를 효율적으로 관리하고 비즈니스 가치를 창출하는 데 최적의 도구가 될 것입니다.

입사 후 첫째, 보유한 Azure 및 Power BI 숙련도를 바탕으로 현업에 즉시 투입되어 데이터 인프라 고도화에 기여하는 '즉시 전력'이 되겠습니다. 둘째, 비즈니스 도메인에 대한 깊은 이해를 바탕으로 데이터 속에 숨겨진 'Why'를 찾아내는 대체 불가능한 전문가로 성장하겠습니다. 끊임없는 학습과 적용을 통해 회사의 성장을 견인하는 핵심 인재가 될 것을 약속드립니다.
