성장 과정 및 데이터 철학

대학 시절 전자공학을 전공하며 접한 캡스톤 디자인 프로젝트는 제 인생의 전환점이 되었습니다. 당시 AI SaaS를 하드웨어에 접목하는 과정에서, 단순히 기술을 구현하는 것을 넘어 "데이터가 어떻게 실질적인 의사결정의 근거가 되는가"에 대한 깊은 매력을 느꼈습니다. 파편화된 로우 데이터가 정교한 분석 파이프라인을 거쳐 비즈니스 가치를 지닌 정보로 변모하는 과정을 목격하며, 저는 데이터 전문가라는 확고한 진로를 설정하게 되었습니다.

이후 저는 독학에 그치지 않고 이론과 실무의 균형을 맞추기 위해 치열하게 노력했습니다. 정보처리기사와 SQLD 자격증을 취득하며 데이터베이스와 소프트웨어 설계의 기초를 다졌고, 서울 ICT 이노베이션 스퀘어의 AI 중급 과정(160시간)을 우수 수료하며 머신러닝의 메커니즘을 이해했습니다. 특히 Microsoft Data School 1기 과정에서의 956시간은 제 역량을 비약적으로 성장시킨 시간이었습니다. Azure 클라우드 환경에서 실제 기업의 데이터를 다루며 ETL 파이프라인 설계부터 시각화까지 End-to-End 프로세스를 완수했고, 이 과정에서 기술적 숙련도뿐만 아니라 '비즈니스 언어'로 데이터를 해석하는 통찰력을 길렀습니다.

저는 단순히 코드를 짜는 개발자가 아닙니다. 데이터 속에 숨겨진 'Why'를 집요하게 추적하여, 기업이 직면한 문제를 해결하고 새로운 성장 기회를 포착하는 전략적 파트너가 되는 것이 저의 지향점입니다. 1,100시간이 넘는 실무 교육과 3건의 대형 프로젝트를 통해 증명한 저의 '즉시 전력감' 역량을 바탕으로 귀사의 데이터 혁신을 주도하겠습니다.

성격의 장·단점

저의 가장 큰 강점은 데이터의 신뢰성을 끝까지 책임지는 꼼꼼함과 근본적인 문제 해결력입니다. 작은 오류가 분석 전체를 왜곡할 수 있기에, 저는 전처리 단계에서 예외 처리 로직 설계에 가장 많은 공을 들입니다. 실제 '인천공항 올인원 서비스' 프로젝트 당시, API 호출 지연 문제를 단순히 수집 주기를 늘려 해결하는 대신 정교한 재시도(Retry) 로직과 모니터링 시스템을 구축하여 파이프라인의 안정성을 99.9%까지 확보한 성과가 있습니다.

반면 완벽주의 성향으로 인해 배포 전 검토 시간이 다소 길어지는 면이 있으나, 이를 보완하기 위해 '우선순위 기반 의사결정 원칙'을 수립했습니다. '비즈니스 가치'와 '기술적 구현 가능성'이라는 명확한 기준을 바탕으로 팀 내 의견을 조율하고 의사결정 속도를 높임으로써, 업무의 완성도와 효율성 사이의 최적의 균형을 유지하고 있습니다.

직무 경험 및 프로젝트

신입이지만 실무에 준하는 End-to-End 데이터 프로젝트를 주도하며 현업 투입 시 즉각적인 성과를 낼 수 있는 역량을 쌓았습니다. Azure 환경에서 데이터 수집부터 시각화까지 이어지는 전체 파이프라인을 직접 구축하고 운영해 본 경험은 저만의 독보적인 강점입니다.

Steam 플랫폼 게임 성공요인 분석

개요: 수십만 건의 게임 리뷰 및 유저 행동 데이터를 분석하여 게임 성공의 핵심 동인을 도출한 B2B 솔루션 프로젝트입니다.

  • 대용량 처리: Selenium과 API를 조합해 약 50만 건의 리뷰 데이터를 수집하고, Databricks 환경에서 메달리온 아키텍처(Bronze/Silver/Gold)를 구축하여 분석 효율을 40% 향상시켰습니다.
  • KPI 발굴: 수집한 로우 데이터를 기반으로 진행한 코호트 분석과 반복검증을 통해 '리뷰 수정율'과 '위험 리뷰 키워드' 등 기존에 없던 새로운 지표를 설계하여 유저 이탈을 사전에 예측할 수 있는 인사이트를 제공했습니다.
  • B2B 대시보드: Power BI를 통해 구현한 '무료 대시보드'와 '유료 대시보드'를 제공, '무료 대시보드'에서 기본적인 정보에 '리뷰 수정율'같은 새로운 지표를 통한 시각화를 제공하여 '유료 대시보드'로 유도, '유료 대시보드'에서는 '유저 이탈 분석','위험 리뷰' 등 세분화된 대시보드를 제공합니다.

지원동기 및 입사 후 포부

데이터는 현대 비즈니스의 '새로운 원유'이지만, 이를 정제하고 가치를 부여하지 못하면 보관도 불편하고 보안 위협까지 있는 위험요소에 불과합니다. 저는 이처럼 가공되지 않은 원석 상태의 데이터를 다듬어, 기업의 성장을 가속하는 '고부가가치 자산'으로 탈바꿈시키는 과정에 큰 자부심을 느낍니다. 방대한 사용자 데이터를 보유한 귀사에서, 제가 쌓아온 Azure 클라우드 엔지니어링 역량과 분석적 통찰력을 발휘하고 싶습니다. 잠들어 있는 데이터를 비즈니스 전략의 핵심 근거로 전환하여, 귀사의 정교한 의사결정 문화를 뒷받침하는 든든한 기술적 토대가 되겠습니다.

입사 후 저의 목표는 명확합니다. 첫째, '즉시 전력'으로서 데이터 파이프라인을 고도화하겠습니다. 1,100시간의 실무 교육을 통해 익힌 Azure 및 Databricks 숙련도를 바탕으로, 현업 투입 직후부터 안정적이고 효율적인 데이터 인프라 운영에 기여하겠습니다. 둘째, 비즈니스 지향적 분석가로 성장하겠습니다. 단순히 숫자를 나열하는 것이 아니라, 이커머스 및 고객 행동 데이터를 심층 분석하여 매출 증대와 사용자 경험 개선을 위한 실질적인 액션 아이템을 매 분기 1건 이상 제안하겠습니다. 끊임없이 학습하고 성과로 증명하여, 3년 내에 귀사의 데이터 전략을 수립하는 핵심 인재로 자리매김하겠습니다.
