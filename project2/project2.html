AI 기반 항공편 지연 정보 알림 서비스

Python SQL Power BI Cosmos DB Azure Stream Analytics Azure ML

기간: 2024. 03 - 2024. 04 (약 1개월)

역할: 데이터 수집, 데이터베이스 구축, 예측 모델 개발 및 배포

프로젝트 목표

  • 예정된 항공편의 지연 여부를 사전에 예측하여 사용자에게 알림 서비스를 제공하는 것을 목표로 합니다.
  • 항공 지연으로 인한 승객의 불편을 최소화하고, 공항 운영 효율을 증대시킵니다.

문제점 및 해결 방안

불만사항

  • 항공편 지연 우려 → 지연 항공 알림 부재
  • 공항까지 소요시간 불확실 → 최적 경로 안내 부재
  • 주차 위치, 잔여석 불확실 → 공항 주차장 정보 부재

필요한 서비스

  • 항공 지연 알림 ➔ 지연 예측 모델 개발
  • 최적 경로 안내 ➔ AI 기반 길안내 서비스
  • 공항 주차장 정보 ➔ 실시간 주차 정보 API 연동

데이터베이스 구축

  • 대량의 실시간 데이터 처리를 위해 **Cosmos DB**를 활용하여 데이터베이스를 구축했습니다.
  • 항공 지연 예측에 필요한 다양한 데이터를 수집하고 정형화했습니다.

모델 개발 및 배포

  • **Azure Stream Analytics**를 사용하여 실시간으로 유입되는 데이터를 처리하고 **Azure Machine Learning**을 활용하여 예측 모델을 개발했습니다.
  • 모델은 항공편 운항 정보, 날씨, 공항 혼잡도 등 다양한 요인을 고려하여 지연 여부를 예측하도록 설계되었습니다.

대시보드

네비게이션 및 실시간 지연 정보

실시간으로 지연 정보를 확인하고, 공항까지의 최적 경로를 안내합니다.

예측 모델을 활용한 지연 알림

항공편 지연 예측 결과와 상세 지연 정보를 제공합니다.

예측 결과에 대한 상세한 분석 자료를 제공하여 사용자가 정확한 정보를 얻을 수 있도록 돕습니다.

문제 해결 과정

1. 데이터 수집 및 정제

문제: 실시간으로 변화하는 항공 데이터를 안정적으로 수집하고 정제하는 것이 어려웠습니다.

해결: 공공데이터포털의 API와 웹 크롤링을 결합하여 데이터를 수집하고, 결측치 및 이상치를 처리하여 데이터의 품질을 높였습니다.

2. 예측 모델 성능 개선

문제: 초기 모델은 예측 정확도가 낮아 실제 서비스에 적용하기에 부족했습니다.

해결: 항공편 지연에 영향을 미치는 다양한 변수(날씨, 공항 혼잡도, 항공기 상태 등)를 추가하고, 여러 머신러닝 모델을 비교하여 최적의 모델(XGBoost)을 선정했습니다.

3. 실시간 서비스 구현

문제: 실시간으로 발생하는 데이터를 예측 모델에 적용하고 사용자에게 즉시 알리는 시스템 구축이 복잡했습니다.

해결: Azure Stream Analytics를 이용하여 실시간 데이터 스트림을 처리하고, Azure Machine Learning에 배포된 모델을 연동하여 예측 결과를 사용자에게 빠르게 전달할 수 있는 파이프라인을 구축했습니다.

기대 효과

  • 사용자: 항공편 지연에 대한 사전 예측 정보를 받아 여행 계획을 유연하게 조정할 수 있습니다.
  • 항공사 및 공항: 지연 예측을 통해 효율적인 자원 배분과 운영 계획을 수립할 수 있습니다.
  • 프로젝트 결과: 데이터 분석, 머신러닝, 클라우드 기술을 통합하여 실제 문제를 해결하는 성공적인 사례가 될 수 있습니다.
