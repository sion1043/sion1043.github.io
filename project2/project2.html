Python SQL Power BI Cosmos DB Azure Stream Analytics Azure ML

역할: 데이터 수집·분석 파이프라인 구축, Power BI 대시보드 시각화

프로젝트 개요

출국 전 불편함을 한 번에 해결할 수 있는 실시간 데이터 기반 앱·웹 서비스 구축

비즈니스 모델

공항, 항공사, 상업시설의 프로모션/광고 연계로 비즈니스 가치 실현

서비스 차별점

  • 다양한 접속환경 및 언어 지원
  • 실시간 대시보드 및 네비게이션 제공
  • 통합 챗봇 서비스 (Azure OpenAI + STT 지원)

프로젝트 내용

  • Google Forms 설문을 통해 실제 사용자 불편 수집 → 서비스 구상
  • 인천공항 API를 Azure Function App(Time Trigger)으로 실시간 데이터 수집
  • Azure Stream Analytics 전처리 후 Cosmos DB 저장
  • Power BI Direct Query로 실시간 대시보드 시각화
  • Azure OpenAI 기반 통합 챗봇 서비스 구현
  • 카카오맵 API + 공항 상업 시설 데이터 → 내비게이션 서비스 구축
  • Azure Static Web/App으로 웹·앱 UI 구현
  • 다국어 지원 (한국어, 영어, 일본어 등)
