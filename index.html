소개

프로젝트

Steam 플랫폼 게임 성공요인 분석 및 시각화

기술 스택: Python · SQL · Selenium · Databricks · Power BI

프로젝트 목적: Steam 플랫폼을 통해 유통되는 다양한 게임 데이터를 분석하고, 성공한 게임들의 특징을 파악하여 게임사들이 자사 게임과의 차이점과 발전 방향을 모니터링할 수 있는 종합 대시보드를 제작.

비즈니스 모델: 무료/유료 모델로 설계하여, 무료 대시보드로 체험 후 더 깊은 인사이트를 원할 경우 유료 서비스로 전환하도록 구성.

프로젝트 내용: Steam API와 크롤링으로 리뷰·게임·유저 데이터를 대량 수집, Databricks 파이프라인으로 수집/전처리 자동화. 분석 후 Power BI로 시각화된 대시보드 제작하여 성공과 실패 패턴을 한눈에 파악 가능하게 구현.

🔗 프로젝트 링크

공항 출국 전 종합 서비스 앱·웹 (ROAD 2 AIR)

기술 스택: Python · SQL · Power BI · Cosmos DB · Azure Stream Analytics · Azure ML

프로젝트 목표: 출국 전의 다양한 불편함을 한 번에 해결할 수 있는 실시간 데이터 기반 앱·웹 서비스 구축.

비즈니스 모델: 공항과 항공사, 상업시설의 프로모션·광고 연계로 비즈니스 가치 실현.

프로젝트 내용: Google Forms 설문으로 니즈 조사 → 데이터 파이프라인 설계 → 인천공항 API와 Azure Function App으로 실시간 데이터 수집 → Azure Stream Analytics로 빠른 쿼리 처리 → Cosmos DB에 저장 및 Power BI 대시보드 시각화 → Azure OpenAI 챗봇과 카카오맵 API를 활용한 길찾기 내비게이션 통합.
