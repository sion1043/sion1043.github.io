소개

프로젝트

Steam 플랫폼 게임 성공요인 분석 및 시각화

기술 스택: Python · SQL · Selenium · Databricks · Power BI

기간

2025. 05 - 2025. 06 (약 2개월)

팀원수 / 역할

6명 / 데이터 수집·분석 파이프라인 구축, Power BI 대시보드 시각화

프로젝트 개요

Steam 플랫폼의 다양한 게임 데이터를 분석하여 성공 요인을 파악하고, 게임사들이 자사 게임의 차이점과 발전 방향을 모니터링할 수 있는 대시보드를 제작

비즈니스 모델

무료/유료 모델로 설계, 무료 체험 후 유료 서비스로 확장 가능

서비스 차별점

단순 리뷰 수집이 아닌 리뷰 수정율, 위험 리뷰, 유저 이탈 분석 등 게임 성공/실패 패턴을 직관적으로 볼 수 있는 지표를 설계

프로젝트 내용

  • Steam API와 크롤링으로 리뷰 · 게임 · 유저 데이터 대량 수집
  • Databricks 파이프라인으로 수집/전처리 자동화 및 메달리온 아키텍처 구성
  • 수집 데이터를 통계 검정, 키워드 분석, 반복 검증으로 분석 후 대시보드 항목 선정
  • Power BI로 시각화된 대시보드 제작(리뷰 수정율, 리뷰 근황, 유저 종합, 리뷰 분류, 위험 리뷰, 유저 이탈 분석, 키워드 트리)

공항 출국 전 종합 서비스 앱·웹 (ROAD 2 AIR)

기술 스택: Python · SQL · Power BI · Cosmos DB · Azure Stream Analytics · Azure ML

기간: 2025. 07 - 2025. 08 (약 1개월)

팀원수 / 역할: 5명 / 데이터 수집·분석 파이프라인 구축, Power BI 대시보드 시각화

프로젝트 개요: 출국 전 불편함을 한 번에 해결할 수 있는 실시간 데이터 기반 앱·웹 서비스 구축

비즈니스 모델: 공항, 항공사, 상업시설의 프로모션/광고 연계로 비즈니스 가치 실현

서비스 차별점: 다양한 접속환경 및 언어 지원과 대시보드, 네비게이션, 통합 쳇-봇까지 사용자의 니즈에 맞는 서비스 다양성. 쳇-봇에 STT(Speech-to-Text) 지원

프로젝트 내용:

  • Google Forms를 통하여 설문을 진행해 실제 사용자들의 출국간에 느끼는 불편을 수집하여 필요한 서비스를 구상하고 전체적인 서비스 파이프라인 설계
  • 인천공항 API을 Azure Function App의 Time Trigger에 의해 실시간 데이터 수집
  • 데이터와 서비스의 실시간 성을 보장하기 위해 Azure Stream Analytics에서 쿼리를 통한 전처리 → Cosmos DB 저장(처리 속도가 빠르고, 갑작스런 처리량 증가에 대응성이 좋음)
  • Power BI 대시보드 시각화 (의미체계의 Direct Query를 사용하여 실시간성 확보)
  • Azure OpenAI를 통해 통합 챗-봇 서비스 구현
  • 카카오맵 API와 공항 내 상업 시설 데이터를 통한 내비게이션 서비스 구현
  • 모든 서비스를 하나의 웹, 앱에서 사용 가능하도록 Azure Static Web, App을 통해 UI 구현
  • 다양한 접속 환경 고려하여 데스크탑, 모바일 환경 구축
  • 언어의 다양성을 고려하여 한국어, 영어, 일본어 등 추가 지원
