소개

데이터 분석가이자 데이터 사이언티스트로서, 데이터를 분석하고 시각화하여 비즈니스 가치를 창출하는 데 목표를 두고 있습니다. 데이터에서 인사이트를 발견하고 이를 기반으로 의사결정을 지원하는 과정에서 즐거움을 느낍니다.

프로젝트

Steam 플랫폼 게임 성공요인 분석 및 시각화

PythonSQLSeleniumDatabricksPower BI

목적Steam 플랫폼을 통해 유통되는 게임들을 모니터링하여 성공 타이틀의 특징을 파악하고, 각 게임사가 자사 타이틀과의 차이점·개선 방향을 찾을 수 있는 종합 모니터링 대시보드 구현.

비즈니스 모델무료·유료 버전으로 분리. 무료로 핵심 지표를 제공하고, 더 많은 지표와 심층 인사이트는 구독 기반 유료 대시보드로 제공.

데이터 수집·처리리뷰(작성일, 추천/비추천 반응, 감성 분석), 게임정보(장르, 순위, 업데이트 일자 등), 유저정보(총 리뷰 수, 보유 게임 수, 플레이 시간 등)를 API+크롤링으로 수집. Databricks 파이프라인으로 수집/전처리/병합 자동화.

분석·대시보드성공/비성공 집단 분류 모델과 지표 산출(리뷰 긍정률, 메이저 업데이트 빈도, 출시 타이밍 등) → Power BI로 비교/추이/랭킹 뷰 제공.

프로젝트 링크

공항 출국 전 종합 서비스 앱/웹 (ROAD 2 AIR)

PythonSQLAzure FunctionsAzure Stream AnalyticsCosmos DBPower BIAzure OpenAI

목표출발 시점의 핵심 불편(출발 시각 결정, 항공 지연, 공항 길찾기, 주차, 빠른 수속 등)을 하나의 서비스로 해결하는 실시간 통합 정보 플랫폼 구축.

데이터 파이프라인인천공항 API(상업시설, 주차 현황, 출국장 혼잡도, 지연·운항 정보, 날씨 등)와 한국수출입은행 환율 API를 Azure Functions로 스케줄 수집 → Azure Stream Analytics에서 실시간 쿼리 → Cosmos DB 저장.

서비스Power BI로 실시간 대시보드(항공편 검색 시 도착지 날씨/상태/환율 연동, 터미널별 혼잡도) 구현. Azure OpenAI 기반 텍스트/음성 챗봇. 카카오맵 API로 공항 내/외부 길찾기 제공.
