프로젝트

기업과 구직자를 연결하는 채용 지원 서비스 (DrawBridge)

기술 스택: Power BI · Python · SQL · RAG · Cosmos DB · Open-Ai · Docker · Azure Migration · Azure Stream Analytics · Azure Function App · React · Tailwind · Cloud Flare

기간

2025. 08 - 2025. 09 (약 1개월)

팀원수 / 역할

5명 / 데이터 수집·분석, RAG 설계·구현, Power BI 대시보드 시각화, 파이프라인 구축

프로젝트 개요

구인난과 구직난이 공존하는 채용 시장의 비효율성을 해소하기 위해, IT 채용 플랫폼 점핏의 데이터를 활용하여 기업과 구직자 간의 간극을 메우는 데이터 기반 채용 지원 서비스입니다. 채용 공고와 구직자 데이터를 분석하고, RAG 기술을 도입하여 구직자의 기술 숙련도를 객관적으로 진단하며, 그 결과를 바탕으로 맞춤형 커리어 가이드와 채용 시장 인사이트를 제공

비즈니스 모델

DrawBridge는 기존 채용 플랫폼인 '점핏'의 부가 서비스 확장을 목표로 함
주요 이해관계자에 따른 비즈니스 모델

  • 점핏 (플랫폼): 서비스 가치 및 활성화 증대를 통한 고객 및 영업 이익 상승.
  • 구직자: 객관적인 숙련도 진단과 맞춤형 커리어 가이드를 통한 취업 효율성 향상.
  • 기업: 지원자 숙련도 데이터 확보 및 공고 분석을 통한 채용 리소스 절약 및 인재 일치도 향상.

서비스 차별점

  • 객관적 숙련도 진단: 주관적이었던 기술 스택 숙련도 평가를 RAG(Azure Open AI, Cosmos DB) 기반의 질문/답변 벡터 유사도 측정을 통해 정량적이고 객관적인 지표로 제공
  • 실시간 데이터 기반: Azure Function App, Event Hub, Azure Stream Analytics를 활용하여 채용 공고 및 기술 트렌드 데이터를 실시간 수집하고 반영
  • 기술 역량 중심 인사이트: 단순히 통계를 나열하는 것이 아니라, 기술 스택 트렌드, 직무별 필요 숙련도 등 구직자와 기업이 실질적인 전략을 세울 수 있는 기술 역량 중심의 심층 인사이트를 Power BI 대시보드로 시각화하여 제공

프로젝트 내용

  • 대규모 데이터 수집 및 더미 생성: 4개 채용 플랫폼, 정부 통계 기반으로 채용 공고 5,308건과 구직자 더미 데이터 10,000건 생성 (Python).
  • 클라우드 데이터 파이프라인 구축: 온프레미스 DB를 Azure PostgreSQL로 마이그레이션하고, Azure Function App 및 Stream Analytics를 활용한 실시간 ELT 파이프라인 구축
  • 메달리온 아키텍처 기반 전처리/분석: Azure Databricks에서 메달리온 아키텍처를 적용하여 데이터를 정제(Silver Layer)하고, 핵심 지표(Gold Layer)를 산출하여 데이터 마트 구축
  • RAG 기반 기술 숙련도 진단 시스템 설계: Cosmos DB(벡터 저장)와 Azure Open AI를 활용하여 스킬별 질문/답변을 생성하고, 사용자 답변과의 COS 유사도를 측정해 객관적 숙련도를 산출하는 API 구현
  • 맞춤형 인사이트 시각화: 분석된 Gold Layer 데이터와 유저 숙련도 결과를 기반으로 구직자에게는 스킬핏/커리어핏을, 기업에게는 채용 트렌드 및 지원자 현황을 제공하는 Power BI 대시보드 개발
  • 웹 서비스 구현: React와 Tailwind를 사용하여 DrawBridge 웹사이트의 UI/UX 및 RAG 연동 기능을 구현

프로젝트 개요

출국 전 불편함을 한 번에 해결할 수 있는 실시간 데이터 기반 앱·웹 서비스 구축

비즈니스 모델

공항, 항공사, 상업시설의 프로모션/광고 연계로 비즈니스 가치 실현

서비스 차별점

다양한 접속환경 및 언어 지원과 대시보드, 네비게이션, 통합 챗봇까지 사용자의 니즈에 맞는 서비스 제공 (STT 지원 포함)

프로젝트 내용

  • Google Forms 설문을 통해 실제 사용자들의 출국 시 불편을 수집 → 필요한 서비스 구상 및 전체 파이프라인 설계
  • 인천공항 API를 Azure Function App의 Time Trigger로 실시간 데이터 수집
  • Azure Stream Analytics에서 전처리 후 Cosmos DB 저장 (빠른 처리·확장성 확보)
  • Power BI Direct Query로 실시간 대시보드 시각화
  • Azure OpenAI 기반 통합 챗봇 서비스 구현
  • 카카오맵 API + 공항 상업 시설 데이터로 내비게이션 서비스 구축
  • Azure Static Web/App으로 웹·앱 UI 구현
  • 다국어 지원 (한국어, 영어, 일본어 등)

Steam 플랫폼 게임 성공요인 분석 및 시각화

기술 스택: Python · SQL · Selenium · Databricks · Power BI

기간

2025. 05 - 2025. 06 (약 1개월)

팀원수 / 역할

6명 / 데이터 수집·분석 파이프라인 구축, Power BI 대시보드 시각화

프로젝트 개요

Steam 플랫폼의 다양한 게임 데이터를 분석하여 성공 요인을 파악하고, 게임사들이 자사 게임의 차이점과 발전 방향을 모니터링할 수 있는 대시보드를 제작

비즈니스 모델

무료/유료 모델로 설계, 무료 체험 후 유료 서비스로 확장 가능

서비스 차별점

단순 리뷰 수집이 아닌 리뷰 수정율, 위험 리뷰, 유저 이탈 분석 등 게임 성공/실패 패턴을 직관적으로 볼 수 있는 지표를 설계

프로젝트 내용

  • Steam API와 크롤링으로 리뷰 · 게임 · 유저 데이터 대량 수집
  • Databricks 파이프라인으로 수집/전처리 자동화 및 메달리온 아키텍처 구성
  • 수집 데이터를 통계 검정, 키워드 분석, 반복 검증으로 분석 후 대시보드 항목 선정
  • Power BI로 시각화된 대시보드 제작(리뷰 수정율, 리뷰 근황, 유저 종합, 리뷰 분류, 위험 리뷰, 유저 이탈 분석, 키워드 트리)
