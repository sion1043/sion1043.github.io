소개

데이터 분석가이자 데이터 사이언티스트로서, 데이터를 분석하고 시각화하며 비즈니스 가치를 창출하는 것에 집중합니다. 데이터 속에서 유의미한 인사이트를 도출하고 이를 기반으로 의사결정을 지원하는 과정에서 깊은 몰입과 즐거움을 느낍니다.

프로젝트

Steam 플랫폼 게임 성공요인 분석 및 시각화

기술 스택: Python · SQL · Selenium · Databricks · Power BI

프로젝트 목적: Steam 플랫폼을 통해 유통되는 다양한 게임 데이터를 분석하고, 성공한 게임들의 특징을 파악하여 게임사들이 자사 게임과의 차이점과 발전 방향을 모니터링할 수 있는 종합 대시보드를 제작하는 것이 목표였습니다.

비즈니스 모델: 대시보드는 무료와 유료 버전으로 구분하였습니다. 게임사들은 무료 버전을 사용해보고 더 많은 데이터와 인사이트를 필요로 할 경우 유료 버전을 구독하도록 설계했습니다.

프로젝트 내용: Steam API와 웹 크롤링을 통해 리뷰 데이터(작성 날짜, 반응, 감성 분석), 게임 데이터(장르, 순위, 업데이트 내역), 유저 데이터(플레이 시간, 보유 게임 수, 리뷰 수)를 수집하였습니다. Databricks의 파이프라인을 통해 데이터 수집·전처리·병합 과정을 자동화하였으며, 분석 결과를 기반으로 성공과 비성공 게임을 분류하고, Power BI로 직관적이고 시각화된 대시보드를 제작하였습니다.

🔗 프로젝트 링크

공항 출국 전 종합 서비스 앱·웹 (ROAD 2 AIR)

기술 스택: Python · SQL · Power BI · Cosmos DB · Azure Stream Analytics · Azure ML

프로젝트 목표: 출국 전 공항 이용자가 겪는 다양한 불편함 — 언제 출발해야 할지, 항공편 지연 여부, 공항 내 길찾기, 주차장 정보, 신속한 수속 등 — 을 한 번에 해결할 수 있는 앱과 웹 플랫폼을 구현했습니다.

비즈니스 모델: 공항에서 제공하는 공식 앱·웹 서비스로, 항공사와의 제휴를 통해 항공권 프로모션과 광고를 진행하며, 공항 내 상업시설과의 협력을 통한 추가 수익 모델을 설계했습니다.

프로젝트 내용: Google Forms 설문을 통해 사용자 불편 사항을 조사하고 이를 기반으로 서비스 파이프라인을 기획했습니다. 인천공항의 다양한 API(상업시설, 주차 현황, 출국장 혼잡도, 항공 운항 정보, 환율 등)를 Azure Function App으로 주기적으로 수집하고, 실시간성이 중요한 혼잡도·지연 데이터는 Azure Stream Analytics로 처리하여 빠르고 안정적인 응답 속도를 구현했습니다. 모든 데이터는 Cosmos DB에 저장해 빠르고 유연한 확장성을 지원했습니다. 또한 Power BI로 실시간 대시보드를 제작하여 항공편 정보, 도착지 날씨, 환율, 출국장 혼잡도, 실시간 주차 현황 등 출국 전 필요한 모든 정보를 한눈에 확인할 수 있도록 구성했습니다. 추가로 Azure OpenAI 기반 챗봇과 카카오맵 API 연동 내비게이션 서비스를 구현하여, 사용자가 텍스트나 음성 입력을 통해 쉽고 빠르게 필요한 정보를 얻도록 했습니다.
