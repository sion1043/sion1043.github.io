프로젝트

Steam 플랫폼 게임 성공요인 분석 및 시각화

기술 스택: Python · SQL · Selenium · Databricks · Power BI

프로젝트 목적: Steam 플랫폼의 다양한 게임 데이터를 분석하여 성공 요인을 파악하고, 게임사들이 자사 게임의 차이점과 발전 방향을 모니터링할 수 있는 대시보드를 제작.

비즈니스 모델: 무료/유료 모델로 설계, 무료 체험 후 유료 서비스로 확장 가능.

프로젝트 내용: Steam API와 크롤링으로 리뷰·게임·유저 데이터를 수집, Databricks로 전처리 자동화, Power BI 대시보드 제작.
