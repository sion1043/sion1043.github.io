소개

프로젝트

Steam 플랫폼 게임 성공요인 분석 및 시각화

기술 스택: Python · SQL · Selenium · Databricks · Power BI

기간

2025. 05 - 2025. 06 (약 1개월)

팀원수 / 역할

6명 / 데이터 수집·분석 파이프라인 구축, Power BI 대시보드 시각화

프로젝트 개요

Steam 플랫폼의 다양한 게임 데이터를 분석하여 성공 요인을 파악하고, 게임사들이 자사 게임의 차이점과 발전 방향을 모니터링할 수 있는 대시보드를 제작

비즈니스 모델

무료/유료 모델로 설계, 무료 체험 후 유료 서비스로 확장 가능

서비스 차별점

단순 리뷰 수집이 아닌 리뷰 수정율, 위험 리뷰, 유저 이탈 분석 등 게임 성공/실패 패턴을 직관적으로 볼 수 있는 지표를 설계

프로젝트 내용

  • Steam API와 크롤링으로 리뷰 · 게임 · 유저 데이터 대량 수집
  • Databricks 파이프라인으로 수집/전처리 자동화 및 메달리온 아키텍처 구성
  • 수집 데이터를 통계 검정, 키워드 분석, 반복 검증으로 분석 후 대시보드 항목 선정
  • Power BI로 시각화된 대시보드 제작(리뷰 수정율, 리뷰 근황, 유저 종합, 리뷰 분류, 위험 리뷰, 유저 이탈 분석, 키워드 트리)

공항 출국 전 종합 서비스 앱·웹 (ROAD 2 AIR)

기술 스택: Power BI · Python · SQL · RAG · Cosmos DB · Open Ai · Azuer Function App · Azure Stream Analytics · Azure ML · Azure Static Web-App

기간

2025. 07 - 2025. 08 (약 1개월)

팀원수 / 역할

5명 / 데이터 수집·분석 파이프라인 구축, Power BI 대시보드 시각화, AI 챗-봇 구현

프로젝트 개요

출국 전 불편함을 한 번에 해결할 수 있는 실시간 데이터 기반 앱·웹 서비스 구축

비즈니스 모델

공항, 항공사, 상업시설의 프로모션/광고 연계로 비즈니스 가치 실현

서비스 차별점

다양한 접속환경 및 언어 지원과 대시보드, 네비게이션, 통합 챗봇까지 사용자의 니즈에 맞는 서비스 제공 (STT 지원 포함)

프로젝트 내용

  • Google Forms 설문을 통해 실제 사용자들의 출국 시 불편을 수집 → 필요한 서비스 구상 및 전체 파이프라인 설계
  • 인천공항 API를 Azure Function App의 Time Trigger로 실시간 데이터 수집
  • Azure Stream Analytics에서 전처리 후 Cosmos DB 저장 (빠른 처리·확장성 확보)
  • Power BI Direct Query로 실시간 대시보드 시각화
  • Azure OpenAI 기반 통합 챗봇 서비스 구현
  • 카카오맵 API + 공항 상업 시설 데이터로 내비게이션 서비스 구축
  • Azure Static Web/App으로 웹·앱 UI 구현
  • 다국어 지원 (한국어, 영어, 일본어 등)
